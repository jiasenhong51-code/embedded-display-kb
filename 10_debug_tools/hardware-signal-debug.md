# 硬件信号调试

## 学习目标

理解显示问题什么时候需要离开软件日志，去观察时钟、同步信号、数据线和接口状态。

## 核心问题

- RGB/LVDS/DSI/HDMI 分别能观察哪些信号。
- pixel clock、hsync、vsync、DE 信号异常会造成什么现象。
- 背光亮但无图、完全不亮、花屏、颜色异常分别可能对应哪些层。
- 示波器、逻辑分析仪在显示调试中如何使用。
- 软件配置正确但屏不亮时，如何排查电源、复位、背光、引脚复用。
