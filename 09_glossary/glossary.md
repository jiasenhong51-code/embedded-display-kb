# Glossary

| Term | Meaning | Notes |
| --- | --- | --- |
| framebuffer | 帧缓冲 | 图像数据本体 |
| plane | 图层 | 可叠加的显示层 |
| crtc | 扫描控制器 | 负责时序与扫描输出 |
| connector | 连接器 | 外部显示链路端点 |
| encoder | 编码/输出编码器 | 连接内部资源与外部链路 |
| DE | Display Engine | 图像合成与处理 |
| TCON | Timing Controller | 面板时序控制 |
| G2D | 2D 图像处理单元 | 缩放、旋转、格式转换 |
